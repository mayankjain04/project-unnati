SmartCity

SmartCity is a collaborative web project created as part of our college minor project. The platform is designed to streamline urban management by focusing on traffic monitoring, garbage disposal, and medical assistance. We are continuously developing new features and improving the current functionality to enhance urban living through technology.

Project Overview:

The current version of SmartCity provides real-time solutions for:

Traffic Monitoring: Live updates on traffic conditions, congestion, and alternate route suggestions.

Garbage Disposal: Track waste collection schedules and optimize garbage disposal with smart notifications.

Medical Help: Provide quick access to emergency medical services, including nearby hospitals and real-time ambulance tracking.

Key Features:

  • Real-time Traffic Monitoring
  • Smart Waste Management
  • Medical Help

Tech Stack:

  • Frontend: HTML, CSS, JavaScript (React.js)
  • Backend: Django
  • Database: MySQL
  • APIs: HERE (for Traffic Monitoring),"https://www.data.gov.in/search?title=garbage" for Garbage Disposal, & unnati api (for Medical Help)
  • Version Control: Git & GitHub
